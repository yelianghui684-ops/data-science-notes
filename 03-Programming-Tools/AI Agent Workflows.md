# AI Agent Workflows

## 一句话概括
把大模型从"聊天"用成"流程":可复用的提示词模板 + 工具调用 + 协作日志。

## 核心要点
- 分层用法:对话(临时)→ 模板(周报 / 研报 / 演讲稿可复用)→ Agent(多步骤自动执行)
- **MCP(Model Context Protocol)**:给模型接上真实工具;跑通一个本地 MCP server 是探索区目标
- 实践沉淀:`~/Documents/Interests and Side/3-AI-Collaboration-Logs/` 已有 presentation 转 speech、社区服务报告等产出
- 纪律:AI 加速信息处理,**结论自己复核**(信息可以让 AI 加速,结论必须自己负责)
- 与本知识库的关系:这个第二大脑本身就是 AI 协作搭建的 → [[🌌 Data Science Second Brain]]
