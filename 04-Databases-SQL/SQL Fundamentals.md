# SQL Fundamentals

## 一句话概括
数据分析师使用频率最高的语言,没有之一。

## 核心要点
- 查询骨架:SELECT … FROM … WHERE … GROUP BY … HAVING … ORDER BY … LIMIT
- JOIN 四兄弟:INNER / LEFT / RIGHT / FULL;笛卡尔积陷阱
- 聚合函数 + GROUP BY;WHERE(行过滤)vs HAVING(组过滤)
- 执行顺序 ≠ 书写顺序:FROM → WHERE → GROUP BY → SELECT
