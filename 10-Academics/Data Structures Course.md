# Data Structures Course

## 一句话概括
2026-04 学期核心课:用 C++ 实现经典数据结构,是算法竞赛与大厂面试的地基。

## 核心要点
- 线性结构:数组、链表、栈 / 队列;哈希表
- 树与堆:二叉树遍历、BST、堆与优先队列
- 图:邻接表 / 矩阵、BFS / DFS
- 每种结构要求:手写实现一遍 + 说清增删查改复杂度
- 桌面有《Data Structure Revision Set.docx》复习卷
