# Competitive Programming

## 一句话概括
洛谷刷题 + C++:算法基本功训练场,目标蓝桥杯 / 校赛。

## 核心要点
- C++ 收口:指针与引用、STL(vector / map / set / string)、RAII
- 算法主题:排序、二分、双指针、递归 / 回溯、DP 入门、BFS / DFS
- 刷题纪律:每周 5-8 题按主题刷(不随机),维护「错题 + 模板」本
- 比赛用固定模板本限时训练,赛后必补不会的题
