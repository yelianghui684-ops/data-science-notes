# Matplotlib与Seaborn

## 一句话概括
Python 静态绘图的标配:Matplotlib 管一切,Seaborn 管好看。

## 核心要点
- Matplotlib 双接口:pyplot(快速)vs 面向对象 `fig, ax`(推荐)
- Seaborn:统计图表一行出——`histplot` / `boxplot` / `heatmap` / `pairplot`
- 与 [[Pandas]] 无缝衔接:`df.plot()`
- 中文显示:配置 `font.sans-serif`
