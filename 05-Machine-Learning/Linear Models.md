# Linear Models

## 一句话概括
可解释、快、稳——永远值得先试的基线模型。

## 核心要点
- 线性回归:最小二乘的解析解与梯度下降解
- 岭回归(L2)/ Lasso(L1):正则化 → [[Optimization]]
- 逻辑回归:sigmoid 输出概率,交叉熵损失
- 系数即解释:业务沟通的利器 → [[Regression Analysis]]
