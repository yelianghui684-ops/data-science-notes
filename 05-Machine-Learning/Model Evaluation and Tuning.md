# Model Evaluation and Tuning

## 一句话概括
不会评估就不会建模——指标选错,努力全废。

## 核心要点
- 分类:准确率的陷阱(类别不平衡)→ 精确率 / 召回率 / F1 / AUC-ROC
- 回归:MAE(稳健)vs RMSE(惩罚大误差)vs R²
- 交叉验证:K-Fold;时间序列要用时序切分
- 调参:网格搜索 → 随机搜索 → 贝叶斯优化(Optuna)
