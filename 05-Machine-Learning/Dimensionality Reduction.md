# Dimensionality Reduction

## 一句话概括
压缩维度、保留信息:既为了可视化,也为了对抗维度灾难。

## 核心要点
- PCA:方差最大方向投影,本质是 SVD → [[Linear Algebra]]
- t-SNE / UMAP:非线性降维,高维数据可视化利器
- 解释方差比:保留多少成分才够
- 注意:PCA 前必须标准化
