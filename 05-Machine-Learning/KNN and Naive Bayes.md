# KNN and Naive Bayes

## 一句话概括
两个"简单却常常够用"的模型:一个靠邻居投票,一个靠概率相乘。

## 核心要点
- KNN:懒惰学习,无训练过程;K 的选择、距离度量、必须标准化特征
- KNN 的维度灾难:高维空间中"最近"失去意义 → [[Dimensionality Reduction]]
- 朴素贝叶斯:特征条件独立假设 + 贝叶斯定理;文本分类经典基线
- 拉普拉斯平滑处理零概率
