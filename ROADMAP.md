# 知识星云 · 更新计划(Roadmap)

> 生成于 2026-07-16,基于当前版本(Three.js + Shader 深空图谱,58 星体)的自主调试结论。
> 原则:每阶段独立可交付,完成后运行 `?test=*` 自测钩子 + 无头截图验收。

## 已完成(v3,当前)

深空渲染(莫兰迪星体 / 微尘视差 / 贝塞尔引力线 / 轨道圈)· 三级 LOD · 视觉隧道 ·
Fly-to(含抽屉偏移)· 详情抽屉 · 小地图(点击 + 拖动)· 浏览航线 · 搜索(回车 / Esc / 失焦)·
深色滚动条 · WebGL 降级 · 内置自测钩子(`?test=hover|drawer|trail|search|filter`、`?view=x,y,k`)

## Phase 1 · 体验完善(优先)

- [ ] **移动端 / 触屏**:双指捏合缩放、单指拖动;窄屏时左面板折叠为悬浮按钮,抽屉改为底部弹出
- [ ] **键盘导航**:`/` 聚焦搜索、`Esc` 关闭抽屉、方向键在相邻星体间跳跃(沿引力线走)
- [ ] **Deep link**:`#note=Pandas` 直达并聚焦星体,便于分享;抽屉打开时同步更新 hash
- [ ] **标签防重叠**:近景下对叶子标签做简单碰撞退避(上下交错)
- [ ] **主题切换**:深空 / 白昼双主题(白昼即此前 Obsidian 风调色板),右上角切换,localStorage 记忆

## Phase 2 · 数据与自动化

- [ ] **GitHub Action 自动构建**:push 时自动运行 `tools/build_graph.py` 并提交 `graph-data.js`,
      彻底免去手动重建;悬空链接检测失败时标红提醒
- [ ] **笔记新鲜度热力**:用 git 最后修改时间给星体加"温度"——近 7 天编辑过的星体微脉冲发光,
      辅助定位"最近在学什么"
- [ ] **图谱统计面板**:孤星(0 连接)清单、连接度 Top10、各领域笔记数,帮助发现知识网络薄弱区

## Phase 3 · 知识导航增强

- [ ] **学习路径**:笔记 frontmatter 增加 `prereq:` 字段 → 近景渲染前置/后置箭头;
      支持"从 A 到 B"最短知识链高亮(BFS 寻路 + 航线动画)
- [ ] **间隔复习模式**:frontmatter `review:` 日期 → "今日待复习"星体聚合成高亮星座,一键逐个飞行
- [ ] **星云雾效**:远景时按领域渲染 GPU 噪声星云团(fragment shader simplex noise),
      取代当前的光晕放大方案

## Phase 4 · 规模化(500+ 星体时再做)

- [ ] InstancedMesh 合并星体 draw call;连线合并为单几何体
- [ ] 四叉树指针拾取(替代 O(n) 最近邻)
- [ ] 布局移入 Web Worker,主线程零卡顿

## 维护须知

- 新增笔记后:`python3 tools/build_graph.py` → push(Phase 2 落地后自动)
- 页面代码全部内联在 `docs/index.html`(ES 模块外链在 `file://` 下会被 CORS 拦截,勿拆分)
- 调参入口:`PALETTE`(配色)、`coreR`(星体半径)、LOD 阈值(0.7 / 2.2)、光晕 `uHalo`(0.17 / 0.28)
