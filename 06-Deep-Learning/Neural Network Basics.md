# Neural Network Basics

## 一句话概括
线性变换 + 非线性激活的层层堆叠 = 万能函数逼近器。

## 核心要点
- 前向传播:矩阵乘法([[Linear Algebra]])+ 激活函数(ReLU / sigmoid)
- 反向传播:链式法则([[Calculus]])自动求梯度
- 损失函数:交叉熵(分类)/ MSE(回归)→ [[Information Theory]]
- 训练技巧:BatchNorm、Dropout、早停、学习率调度
