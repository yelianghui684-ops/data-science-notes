# 🌌 数据科学第二大脑

> 这是知识库的中心。所有领域从这里辐射出去,形成一张星状知识网络。

## 九大领域

- [[数学基础 MOC]] — 线性代数、微积分、最优化
- [[统计学 MOC]] — 概率、推断、实验设计
- [[编程与工具 MOC]] — Python、NumPy、Pandas、Git
- [[数据库与SQL MOC]] — SQL、数据建模、NoSQL
- [[机器学习 MOC]] — 监督/无监督、模型评估、特征工程
- [[深度学习 MOC]] — 神经网络、CNN、Transformer、LLM
- [[数据可视化 MOC]] — 图表设计、交互式可视化
- [[数据工程 MOC]] — ETL、大数据、MLOps
- [[项目实战 MOC]] — Kaggle、端到端项目

## 使用方法

1. 每个领域有一个 **MOC(Map of Content)** 枢纽笔记,汇总该领域的所有主题
2. 新建笔记时,务必用 `[[双链]]` 挂到对应 MOC 上,保持星状结构不散
3. 用 [[笔记模板]] 创建新笔记
4. 三维星状图:仓库 `docs/index.html`,推送后可通过 GitHub Pages 在线查看
