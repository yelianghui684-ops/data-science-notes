# MLOps

## 一句话概括
让模型从 notebook 走进生产并持续可靠:训练、部署、监控的工程化。

## 核心要点
- 实验跟踪:MLflow / W&B 记录参数、指标、模型版本
- 部署形态:批量预测 / REST API(FastAPI)/ 流式
- 监控:数据漂移、概念漂移、性能衰减
- CI/CD for ML:代码、数据、模型三者都要版本化 → [[Git and Version Control]]
